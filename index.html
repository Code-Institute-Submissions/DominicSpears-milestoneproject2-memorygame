How to play!

The aim of the game is to reveal all the cards on the grid by matching pairs. Matched pairs will remain face up while mis-matched pairs will flip back. Try to remember where you've seen a card image before to speed up your success rate. Reveal all cards within the time limit to win

Reset

If at any stage you want to restart the game and begin again, click the reset button in the navigation menu.

Memory Games

Playing memory games can improve the functions of the brain, including attention, concentration, and focus. By improving these functions you can improve your short term memory and make it easier to learn new skills.

Memory games can improve visual recognition and visual discrimination. Playing memory games based on spotting differences, or linking two related images, leads to an acceleration in processing visual stimuli.

Memory games can aid the healthy functioning of the brain and help to prevent diseases that reduce cognitive function such as dementia
